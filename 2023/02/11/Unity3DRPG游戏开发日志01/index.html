Unity3DRPG游戏开发日志01

发表于2023-02-11|更新于2023-02-14|Unity
|字数总计:1160|阅读时长:4分钟|阅读量:

1. Create Project 创建项目导入素材

1.1 创建渲染管线，升级到URP项目

点击菜单栏中的 window → package manager → 左上角 packages:in project 切换成 unity registry
搜索 Universal RP → install

在 project 窗口上点击加号 → Rendering → Universal Render Pipeline
→ Pipeline Asset(Forward Renderer) → 回车

新建文件夹，起名 Pipeline Settings，把将刚刚创建的渲染管线存放进去方便管理

1.2 设置渲染管线

点击菜单栏中的 Edit，打开 Project Settings，点 Graphics
可以看到 scriptable render pipeline settings 为空，点击右边的小圆点，将刚刚创建好的 URP 放到这里

点 Quality，不同分辨率下可以根据需要选择用不同的渲染管线，在 ultra 下点 rendering 右边的小圆点，选择刚刚创建的 UPR

1.3 导入素材

去网页的Unity商店中购买需要的素材

  • 要注意素材是否支持通用渲染管线 URP 或 lightweight RP （ LWRP ）

点击菜单栏中的 window → package manager → 左上角 packages:in project 切换成 my assets
找到刚刚购买的资源，下载并导入

创建文件夹 Assets Packs，把导入的素材放进去统一管理

在人物素材 DogKnight 里找到 Prefab 文件夹，把 dogpolyart 放进场景里

打开场景素材包 SimpleNaturePack，双击 SimpleNaturePack_2020.3_URP_v1.23 再次导入

edit → render pipeline → universal render pipeline → upgrade material to universalRP materials
将整个项目材质升级到 URP ，然后人物就由粉色状态变成正常显示

2. Build Level 尝试熟悉基本工具

2.1 设置通用渲染管线

下载并导入购买的 skybox

window → rendering → lighting
打开光源窗口，选择 environment，在 skybox material 里可以看到目前是默认的天空盒子，点右边的小圆点可以切换成自己导入的

在 project 里找到并点击刚刚自己选择的天空盒子，在 inspector 面板可以更改数值，调整自己喜欢的颜色

在场景素材里找到 ground 并拖入场景内，将人物y轴设置成 0.5 以便让他完整显示

找到自己创建的渲染管线，可以看到右边有渲染的质量，光线，和阴影等可以调整

  • Max Distance 可以调整阴影的渲染距离，默认设置为 50，意思是 50 米以内的物体都进行阴影的渲染
  • Cascade Count 将阴影渲染距离进行分段处理，实现在近处阴影清晰，在远处阴影模糊
  • 调整 Normal Bias 可以将阴影的缝隙给削去
  • Lighting 中的 Resolution 调整光线的精度
  • HDR 在 Post processing 后处理的时候会用到，anti aliasing（msaa） 抗锯齿可以选择开或者不开
  • soft shadows 勾选后可以让阴影变得更虚
  • normal bias 数值越小，阴影缝隙越少

2.2 场景光照设置

window → rendering → lighting
点击 lighting 下面的 scene， 点 new lighting settings 创建一个新的光源设置，将他重命名

这是个准对某一场景进行光线设置的文件

lighting mode 通常选择 baked indirect 间接烘焙

在 lightmapper 处选择 progressive GPU(preview),这样可以节省 CPU 开销

点最下方 generate lighting 渲染光线

切换回 environment，在 environment lighting 下的 source 处可以选择光线的颜色，默认是 skybox，也就是天空盒子的颜色，根据需要进行修改

2.3 场景设计技巧

  • 选中预制体，拖拽进场景里，按住V键后把鼠标移动到预制体的某个点上，按住左键进行移动，这样预制体会自动吸附在另一个物体上
  • 选中Main Camera, 使用 Ctrl + Shift + F 将 GAME 的视角切换为 scene 的视角
  • 在 samplescene 窗口右键点击 create empty，创建一个空的gameobject 进行物品归类，将其重命名为 environment ，reset 他的坐标，把树、石头等物品放入
  • 创建空的 gameobject，重命名为 characters，把人物放在他的下面，不成为子集，这样可以让场景分类，区分不同的物体
