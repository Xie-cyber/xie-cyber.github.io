Unity游戏开发准备工作02

发表于2023-02-14|更新于2023-02-17|Unity
|字数总计:1028|阅读时长:3分钟|阅读量:

1. PolyBrush 发挥创意构建场景

1.1 使用 Poly Brush来构建 low poly的terrain场景

在 window → package manager 中找到 polybrush 进行安装

安装好后展开 samples，导入有 universal RP 的 shader examples

新建文件夹 Addons 将 Ploybrush 这些 package 插件装进去

菜单栏 Tools → polybrush → polybrush window

polybrush 上有五个菜单

  1. 用来调整地面的高矮
  2. 用于柔化
  3. 将当前物体刷上不同的颜色
  4. 将预制体刷到场景当中
  5. 将 Taxtrue 刷到场景当中,可以将草地或者地面等素材直接应用到当前场景选中的物体

菜单 1

  • 可以看到 low poly 的图形低模的顶点
  • 按左键向上凸起，按 ctrl 再按左键向下凹陷
  • 按 CTRL 调外圈大小，按 shift 调内圈大小
  • 调整 direction 改变凸起方向
  • brush mirroring 能复制笔刷，做到一次性刷两个地方

菜单 3

  • 需要 polybrush 自带的 shader graph 创建材质
  • 在polybrush 的 shader graph 右键，新建 material，把他拖拽到地面上即可应用
  • 选择某颜色，点 flood 把颜色应用到整个场景

菜单 4

  • 选择需要的预制体，拖拽到 current palette
  • 选中需要的预制体，在场景中按住左键并移动，随机出现预制体
  • 拖拽 brush loadout 修改出现概率
  • 画好后按 CTRL 再按左键移动就可删除
  • 注意勾选 Hit Surface is Parent，这样刷出来的物体全都会被囊括到同一子集下，更加方便管理

1.2 使用Probuilder创建一个非常大的平台

在 package manager 搜索并安装 Probuilder

在 samples 处导入 URP 支持

在Tool菜单里 面吧Probuilder的窗口调出来放到界面上

点右侧三个点，选 use icon mode 可以改成 icon 模式，按 shift 键可以看到详细说明

按住 alt 键，在 probuilder 选中第一个 icon，可以看见地图上出现一个物体，以及弹出来的 shape tool 窗口

在 shape tool 窗口上的 shape selector 处将 cube 改成 plane

修改长宽和顶点数量，点击 build 创建物体

点击刚刚创建的 plane，在 probuilder 窗口选择 center pivot，可以把坐标锚点改到物体中心

1.3 使用 Preview Package 的 ProGrid 来查看效果

Preview package 的选项没有打开，需要手动打开（可能会造成一些问题，比如项目崩溃）

Edit → Project Settings → Package Manager → 勾选 Enable Preview Packages

在 Package Manager 中安装好 Progrids 后，打开，就可以看到左上角的 Grids 按钮了

probulid 点 triangulate objects 将 Plane 的分割改为三角的，更符合 low-poly 风格

新建文件夹 Materials，新建材质球 Ground，拖拽到地面上，任意改变颜色

2 智能导航地图烘焙

window → AI → Navigation

选中地面，在 inspector 右侧打开 static 下拉菜单选择 Navigation Static

在 Navigation 窗口 object 中的 navigation area 选择 walkable，将这些地面设置为可行区域

进入 Navigation 的 bake 菜单，点击烘焙，可行区域会变成蓝色

在 navigation 的 bake 菜单中，调整 max slope，通常设置30-45左右，让游戏人物不能前往倾斜角度太大的地方，包括深坑和山坡

选中所有的树，在 navigation 的object 处，勾选 navigation，把 navigation 设置成 not walkable，在 bake 菜单再次烘焙

选中小狗，来设置 Agent，可以看到小狗身上有一个 Component 是 Animator，其中的绑定了一个 Animation 是展示该模型所有的动作，可以暂时移除

在 Inspector 窗口中给小狗添加一个 Component，Nav Mesh Agent

在 Nav Mesh Agent 组件的位置，可以先把 obstacle avoidance 处的 height 和 radius 调小一点，让圆柱体和模型差不多高矮

然后在 navigation 的 bake 菜单填写一样的高度和宽度，再次烘焙，可以看到树周围禁止移动的范围变小了

选择一块石头，添加 nav mesh obstacle 组件，把 shape 设置为 capsule，也可以默认为 box

添加组件的好处是，假设该石头在游戏里实时移动了，也可以拥有禁止移动的区域，防止玩家穿模
